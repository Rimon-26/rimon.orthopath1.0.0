# অর্থপাঠ (Orthopath)

ফিন্যান্স, ব্যাংকিং ও বিমা — একাদশ/দ্বাদশ শ্রেণীর শিক্ষার্থীদের জন্য শিক্ষা অ্যাপ (PWA)।

## GitHub Pages এ Deploy করার নিয়ম (মোবাইল থেকে)

1. GitHub এ একটা নতুন repository বানান, নাম দিন যেমন `orthopath` (public রাখবেন)
2. এই ফোল্ডারের সব ফাইল (index.html, manifest.json, sw.js, css/, js/) repository তে আপলোড করুন
   - GitHub অ্যাপ বা মোবাইল ব্রাউজার দিয়ে GitHub.com এ গিয়ে "Add file" > "Upload files" ব্যবহার করুন
3. Repository এর **Settings > Pages** এ যান
4. Source: **Deploy from a branch** সিলেক্ট করুন, branch: **main**, folder: **/ (root)** — Save করুন
5. কিছুক্ষণ পর আপনার অ্যাপ চালু হবে এই লিংকে:
   `https://<আপনার-ইউজারনেম>.github.io/orthopath/`

## এডমিন লগইন

- এডমিন প্যানেলে ঢোকার পাসওয়ার্ড: `Orthopath@Rimon26`
- এটা বদলাতে চাইলে `js/app.js` ফাইলের একদম উপরে `ADMIN_PASSWORD` ভ্যারিয়েবলটা এডিট করুন

## গুরুত্বপূর্ণ সীমাবদ্ধতা (এখনকার ভার্সনে)

- সব ডেটা প্রতিটি ফোনের **localStorage**-এ আলাদাভাবে জমা থাকে (অফলাইন-ফার্স্ট)
- একজন ছাত্রের রেজাল্ট অন্য ফোনে/আপনার এডমিন ফোনে দেখা যাবে না, যতক্ষণ না Firestore যোগ হয়
- ব্রাউজার ডেটা/ক্যাশ ক্লিয়ার করলে বা অ্যাপ আনইনস্টল করলে ডেটা হারিয়ে যেতে পারে
- এই সীমাবদ্ধতা দূর করতে Firestore + ছাত্রদের আসল অ্যাকাউন্ট সিস্টেম দরকার হবে
